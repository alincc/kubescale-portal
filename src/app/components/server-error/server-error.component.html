Server Error

There seems to be a problem with the server connection!

If not done already, consider following the following steps:

  • Download XAMPP from https://www.apachefriends.org/download.html (with PHP).
  • Start up the Apache server from the XAMPP Control Panel.
  • Clone the dummy back-end repository git clone https://github.com/maxkha-3/kubescale-backend-dummy.git into the "C:/xampp/htdocs" directory.
  • Refresh the application and connection should be established.

Application will retry the connection once every 5 seconds and redirect to home, when server is back up.
